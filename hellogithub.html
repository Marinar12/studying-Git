C помощью Git программисты и разработчики ориентируются в коде и отслеживают изменения. Git помогает вернуть файлы в исходное состояние и видеть изменения, внесенные в определенный период. Разработчк выполняет разные команды(например, commit,push), а все изменения синхронизируются с центральным репозиторием. Git - это система конторля версий, а GitHub - онлайн-сервис, по сути социальная сеть. Одна из основных целей GitHub - быть единым местом для проектов с исходным кодом. Предполагается, что пользователь делится чем-то полезным, а другие люди смогут учавствовать в разработке.
